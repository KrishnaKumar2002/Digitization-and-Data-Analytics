# Digitization, Data Analytics, Machine Learning, and Deep Learning Workspace

This repository is my study and practice space for **digitization, data analytics, machine learning, and deep learning**.  
It collects lecture material, reference books, datasets, notebooks, and exercise submissions used across the course work.

## What This Folder Contains

- `DDA EXERCISE/` - lecture exercises, notebook work, preprocessing tasks, and machine learning assignments
- `DAA/` - additional statistics and preprocessing practice files
- `MIT DL Books/` - deep learning book resources and chapter-wise PDFs
- `*.pdf` - lecture notes, research papers, and reference books
- `*.ipynb` - Python notebooks for statistics, preprocessing, clustering, regression, and other analytics tasks
- `*.csv`, `*.txt`, `*.png`, `*.html` - supporting datasets, instructions, and visual/material assets

## Topics Covered

### Digitization
- Data handling and structured digital workflows
- Practical file-based exercises and notebook-based learning

### Data Analytics
- Descriptive statistics
- Data preprocessing
- Data management concepts
- Exploratory analysis and interpretation

### Machine Learning
- Similarity and clustering
- Decision trees
- Naive Bayes classification

- Model evaluation and basic implementation practice

### Deep Learning
- Core deep learning concepts
- Reference reading from the MIT Deep Learning Book
- Foundational material for neural networks and modern learning systems

## Folder Highlights

- `DDA EXERCISE/01-Introduction/` - introductory material

- `DDA EXERCISE/03-BasicsOfStatistics/` - statistics notebooks and datasets
- `DDA EXERCISE/04-Machine Learning-I/` - early ML exercises and clustering work
- `DDA EXERCISE/Exercises/` - organized exercise copies for practice and submission
- `DDA EXERCISE/05-MachineLearning-II/` - classification and clustering notebooks

## Purpose

The goal of this workspace is to keep all learning material in one place and track my hands-on practice across the full pipeline:

1. Understanding the data
2. Cleaning and preprocessing it
3. Applying statistical methods
4. Building ML models
5. Studying deep learning foundations
6. Reviewing results and improving the workflow

## Notes

- Most files are course or self-study resources rather than a single runnable application.
- Notebook outputs, datasets, and reference PDFs are intentionally kept together for learning and revision.
- Some folders contain duplicate exercise copies to support multiple practice versions.

## Quick Start

If you are exploring this repository for the first time:

1. Start with the introduction and preprocessing notebooks.
2. Review the statistics notebook in `03-BasicsOfStatistics`.
3. Move to clustering, classification, and regression exercises.
4. Use the PDFs in the root and `MIT DL Books/` as reference material.
